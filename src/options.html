Getting a Property ID

  1. Browse to Google Analytics.
  2. Select Admin » the account of your choice » New Property.
  3. Choose Web Site, and enter any Name and URL (we recommend 'Personal Web Analytics' and google.com respectively).
  4. Save the Tracking ID (sometimes called a Property ID) into settings.

Setup Google Analytics

Browse to your new Properties settings, in Site Search Settings, select Do track Site Search » Query parameter and set it to q.
